# 激突 ミッドウェー ― テレビ特番風 3D俯瞰デモ

昭和17年(1942年)6月4日(現地時間)のミッドウェー海戦を、テレビ特番のCG再現映像風に
自動再生する Nuxt + Three.js アプリケーション。関ヶ原版(`sekigahara-3d`)と同一の演出フレームワーク。

構成要素: 波シェーダの太平洋・ミッドウェー環礁 / 3艦隊29隻のプロシージャル艦船(空母・戦艦・重巡・駆逐艦) /
8個飛行隊のV字編隊飛行(発艦・攻撃・被撃墜) / 攻撃経路矢印 / 爆発・水柱・黒煙・火災 /
雲と夜明け→白昼→日没の天候変化 / キーフレームカメラワーク

## 起動

```bash
npm install
npm run dev   # http://localhost:3000
```

## 番組構成(全150秒・自動ループ)

| 章 | 現地時間 | 内容 |
|---|---|---|
| 序章 | 午前4時30分 | 暁の出撃。南雲機動部隊、友永隊108機発艦 |
| 第一章 | 午前6時20分 | ミッドウェー空襲。「敵空母見ユ」の打電 |
| 第二章 | 午前9時20分 | VT-8雷撃隊全滅。零戦が低空に釘付けに |
| 第三章 | 午前10時20分 | 運命の五分間。加賀・蒼龍・赤城被弾炎上 |
| 第四章 | 正午〜 | 飛龍の反撃、ヨークタウン大破 → 飛龍被弾 |
| 終章 | 午後5時〜 | 機動部隊壊滅、日没の撤退 |

## 操作

- **▶ / ❚❚** 再生・一時停止、**↺** 最初から、**×1** 速度切替(1x/2x/0.5x)
- シークバーでスクラブ、章ボタン(序〜終)でジャンプ
- デバッグ用URLパラメータ: `?t=75`(75秒地点から開始)、`&paused`(一時停止で開始)

## 構成ファイル

- `app/utils/navalData.ts` … 艦隊布陣・飛行隊経路・爆発・カメラワーク・章構成のすべての演出データ
- `app/components/NavalScene.vue` … Three.js シーン(海シェーダ・艦船・編隊・矢印・爆発・雲)
- `app/app.vue` … 番組風オーバーレイUI(ロゴ・章テロップ・速報・現地時間・コントロール)

演出の調整はほぼ `navalData.ts` のキーフレーム編集だけで完結します。
